# Memu AI

![Memu AI cover](assets/readme-cover.svg)

Memu AI is an iOS chat app built with SwiftUI. The project focuses on conversation history, clean settings screens, native navigation, and a simple Xcode-first structure.

## Run

```bash
git clone https://github.com/mertefekurt/Memu-AI.git
cd Memu-AI
open Memu.xcodeproj
```

Select the `Memu` scheme in Xcode, choose an iOS simulator, and run.
